# Product Vision: ForgeFlight
ForgeFlight is an AI-powered "Cognitive Flight Deck" designed as a premium study aid to help student pilots master ground school and aircraft systems.

## Target Audience
Version 1.0 is a private application built for a 16-year-old student pilot training in a Cessna 172 at Elite Aviation. The tone must be encouraging, deeply technical but explained in relatable ways (using analogies like cars/sports), and relentlessly focused on safety.

## Core Features (MVP)
1. **Secure Access:** A simple login screen so only the student can access the dashboard.
2. **The Flight Deck UI:** A mobile-friendly dashboard with a chat interface.
3. **Quick Action Modes:** Buttons above the chat to trigger specific study modes: "Translate METAR", "Armchair Flight Scenario", and "Grumpy DPE (Oral Exam Prep)".

## Core Rule
The AI must NEVER hallucinate aviation data. All speeds, limitations, and procedures must be strictly based on the Cessna 172 and Elite Aviation data.
